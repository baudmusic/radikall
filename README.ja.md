## はじめに

山下達郎の楽天カード サンデー・ソングブックをもっと快適に聴きたくて、このアプリを作りました。もうすぐ夏ですね。Big Wave を聴く季節です。

ついでに言わせてもらえれば、Claude は本当にケチです。使用量の上限があっという間に消えていきます。

## 概要

Radikall は、Radiko の公式 47 都道府県のすべての日本の放送局をカバーするクロスプラットフォームのラジオ クライアントであり、完全な局のブラウジングと再生をサポートする Radiko のネイティブ クライアントを備えており、ネットワーク接続があれば世界中のどこからでも聴くことができます。

スムーズな局ブラウジング、現在再生中の詳細表示、週間番組表、Windows でのトレイ再生コントロール、そして Windows と Android にまたがる多言語設定システムを提供します。

本プロジェクトは [jackyzy823/rajiko](https://github.com/jackyzy823/rajiko) のアイデア・リバースエンジニアリングの手法・インスピレーションによって実現しました。あのリポジトリなしには、このネイティブアプリの実装ははるかに困難だったでしょう。

## 使用前に

ソースからビルドするのではなく、アプリをそのまま使いたい場合は、まずこのセクションをお読みください。

### Windows

1. GitHub Releases ページから最新の Windows インストーラーをダウンロードしてください。
2. 先に **VLC メディアプレーヤー**をインストールしてください。
3. VLC をインストールしてから Radikall を起動してください。デスクトップの再生は `vlcj` 経由の `libVLC` に依存しています。

推奨環境：

- Windows 10 または Windows 11
- 64 ビットシステム
- 安定したインターネット接続

### Android

1. GitHub Releases ページから最新の**署名済み** Android APK をダウンロードしてください。
2. Android が確認を求めた場合は、不明なソースからのインストールを許可してください。
3. アプリを開き、再生前にエリアを選択してください。

推奨環境：

- Android 7.0 以降
- 安定した Wi-Fi またはモバイル回線

### ソースからビルド

プロジェクトを自分でビルドしたい場合は、以下をインストールしてください：

- Git
- 完全な JDK 24 のインストール
- Android SDK 35 を含む Android Studio
- Platform Tools / `adb`
- デスクトップ再生テスト用の Windows 上の VLC メディアプレーヤー

その後、以下を実行してください：

```powershell
.\gradlew.bat :desktopApp:run
.\gradlew.bat :androidApp:installDebug
```

## リリースビルド

### Windows

```powershell
.\gradlew.bat :desktopApp:packageReleaseExe :desktopApp:packageReleaseMsi
```

出力先：

- `desktopApp/build/compose/binaries/main-release/exe/`
- `desktopApp/build/compose/binaries/main-release/msi/`

### Android

```powershell
.\gradlew.bat :androidApp:assembleRelease
```

出力先：

- `androidApp/build/outputs/apk/release/`

注意：

- このプロジェクトは現在、**1 つのユニバーサル Android APK** をビルドします。ABI 分割や密度分割は有効にしていません。
- Windows では **2 種類のインストーラー形式**（`EXE` と `MSI`）が生成されます。
- Windows パッケージングには **`jpackage` を含む完全な JDK** が必要です。一部の IDE にバンドルされた簡易版 JetBrains Runtime では動作しません。
- `assembleRelease` は現在 `androidApp-release-unsigned.apk` を生成します。ユーザーに配布する前に、自分のキーストアで署名する必要があります。

## 技術スタック

Radikall は Kotlin Multiplatform アーキテクチャで構築されています：

- `shared/`：共有ビジネスロジック、Compose UI、テーマシステム、ローカライズ、設定、局データ、再生状態、番組表ロジック
- `androidApp/`：Android ホストアプリ、`Media3 ExoPlayer`、ライフサイクルフック、アラーム、バックナビゲーション、APK パッケージング
- `desktopApp/`：Compose Desktop シェル、カスタムタイトルバー、トレイ統合、Windows パッケージング、デスクトップウィンドウの動作

リポジトリで使用するコア技術：

- Kotlin 2.1.0
- Compose Multiplatform 1.7.3
- Material 3
- Ktor 3
- kotlinx.serialization
- kotlinx.datetime
- Coil 3
- Android `Media3 ExoPlayer`
- デスクトップ `vlcj` + ローカル HLS プロキシによる再生
- JVM Preferences / Android 永続化設定
- 簡体字中国語・繁体字中国語・英語・日本語・韓国語対応のアプリ側多言語システム

## デザイン言語

このカラーパレットは偶然ではなく、意図的に選ばれたものです。

- `#D0104C` は**韓紅花 / karakurenai**
- `#005CAF` は**瑠璃 / RURI**
- `#113285` は**紺青 / konjyo**

**韓紅花 / karakurenai** は、染料・織物・儀式・東アジアの伝統的な色名と結びついた、鮮やかな歴史的な赤の感覚を持ちます。平安時代、この極めて彩度の高い深紅を染め出すには、莫大な量の紅花（サフラワー）の花びらを要し、その費用は非常に高価なものでした。そのため、最高位の貴族の富と権力の象徴となり、かつて日本の朝廷によって庶民の使用を絶対に禁じた「禁色」に定められました。百人一首の在原業平の名歌「ちはやぶる神代もきかず龍田川からくれなゐに水くくるとは」は、まさにこの韓紅花の絢爛さと濃烈さを借りて、川面を染め尽くす紅葉の圧倒的な美しさを詠んでいます。

**瑠璃 / RURI** は仏教の「七宝」のひとつに数えられます。病を癒し寿命を延ばす仏・薬師如来が住まう東方の浄土は「東方浄瑠璃世界」と呼ばれます。そのため、この色は日本の歴史を通じて、神聖・清浄・無垢、そして神秘的な宗教的力と結びついてきました。古代の日本ではこの色の原料を産出することができず、すべてが険しいシルクロードを渡って運ばれてきました。奈良の正倉院には聖武天皇の時代の「瑠璃杯」が今も完全な姿で保存されており、当時それは単なる色ではなく、遠い西域と極楽浄土への想像を体現するものでした。

**紺青 / konjyo** は織物・絵具・夜・海・遠方の深い青をもたらします。古来の日本画において、紺青は藍銅鉱（アジュライト）を磨り潰して作られた伝統的な天然鉱物顔料です。職人は鉱石を砕いた後、水簸法（すいひほう）で分離し、最も粒子が粗く色の深い層を紺青として取り出しました。飛鳥・奈良時代の古墳壁画（高松塚古墳など）や格式高い屏風絵に広く用いられていました。江戸時代後期、ヨーロッパで発明された合成顔料「プルシアンブルー」が日本に伝わり、「紺青」あるいは「ベロ藍」と呼ばれるようになりました。この新しい紺青は安価で発色が鮮やかなうえに褪色しにくく、従来の青色顔料の制約を一気に打ち破りました。葛飾北斎はこの新しい「紺青」を大量に用いて世界を驚かせた《神奈川沖浪裏》を生み出し、浮世絵風景画（名所絵）の全面的な隆盛を直接後押ししました。

この3色はいずれも「日本と世界のつながり」を象徴しています。このアプリでこれらの色を選んだのは、シンプルなことを伝えたかったからです：ラジオは、人よりも海を越えやすい。
