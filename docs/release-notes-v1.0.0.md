# iOS WebApp 1.0.0

## 中文

Web App：
[https://radikall-web.baudmusic.workers.dev](https://radikall-web.baudmusic.workers.dev)

这个版本把 Radikall 的 iOS Web App 正式公开上线。现在可以直接用 Safari 打开一个稳定网址，然后保存到主屏幕或桌面使用，不需要经过 App Store。

更新亮点：

- iPhone / iPad 可直接保存到主屏幕使用
- macOS 可直接用 Safari 打开并保存到桌面使用
- 使用稳定的公网地址，不需要下载压缩包
- Web UI 与 Android 版保持接近的结构与交互

使用方式：

- iPhone / iPad：用 Safari 打开这个网址，然后使用“分享 -> 添加到主屏幕”
- macOS：用 Safari 打开这个网址，然后保存到桌面
- 其他设备：也可以直接在浏览器中打开同一个网址

说明：

- iOS / macOS 端推荐使用 Safari
- Windows 与 Android 原生版本仍然通过 GitHub Releases 提供

## English

Web App:
[https://radikall-web.baudmusic.workers.dev](https://radikall-web.baudmusic.workers.dev)

This release officially publishes the Radikall iOS Web App. It can now be opened directly in Safari and saved to the Home Screen or desktop, without going through the App Store.

Highlights:

- iPhone / iPad support through Safari + Add to Home Screen
- macOS support through Safari + desktop save flow
- One stable public URL, with no archive download required
- Web UI brought closer to the Android app structure and interaction model

How to use:

- iPhone / iPad: open the URL in Safari, then use Share -> Add to Home Screen
- macOS: open the same URL in Safari, then save it to the desktop
- Other devices: open the same URL directly in the browser

Notes:

- Safari is the recommended browser for iOS and macOS
- Native Windows and Android builds are still distributed through GitHub Releases
